# GRVEZ VAULT

**A living vault of memory, noise, and ritual-coded visuals by Sheldyn Gruhn.**

GRVEZ VAULT is a dark, immersive portfolio site showcasing artistic works as relics stored in a vault rather than displayed in a gallery. The design emphasizes fragmentation, distortion, and the coexistence of multiple signal channels: Gizzy Graves, DMONIX, and The Murmur.

---

## 🚀 Quick Start

### Option 1: GitHub Pages (Fastest)
1. Go to **Settings → Pages**
2. Select `main` branch as source
3. Save and wait 1-2 minutes
4. Your site is live at: `https://Smgruhn.github.io/grvezvault/`

### Option 2: Network Solutions FTP
1. Download repo as ZIP
2. Extract all files
3. Log into Network Solutions control panel
4. Use File Manager or FTP client to upload:
   - `index.html`
   - `styles.css`
   - `script.js`
5. Point domain DNS to Network Solutions
6. Access at your custom domain

### Option 3: Webflow Integration
1. Create new Webflow project
2. Import the HTML/CSS
3. Customize in Webflow's visual editor
4. Deploy to custom domain

## 📝 Customization

### Update Content
Edit `index.html` and replace:
- `PROJECT TITLE` with your actual work titles
- `Relic 01`, `02`, `03` with your archive entries
- Email placeholder with actual contact info
- Description text in hero and sections

### Customize Colors
Edit CSS variables in `styles.css` (`:root`):
```css
--accent: #ff1744;        /* Primary highlight color */
--bg: #050509;            /* Dark background */
--fg: #f5f5f7;            /* Foreground text */
--muted: #9a9aa5;         /* Secondary text */
```

### Add Images
Replace placeholder divs with actual images:
```html
<!-- Replace this: -->
<div class="thumb-placeholder">HERO / COVER IMAGE</div>

<!-- With this: -->
<img src="your-image.jpg" alt="Project title" style="width: 100%; height: 100%; object-fit: cover; border-radius: var(--radius-md);">
```
